## What's New

### Voice Gateway

Full implementation of `coregateway.Gateway` interface for real-time voice AI:

```go
import "github.com/plexusone/omni-telnyx/omnivoice/gateway"

gw, err := gateway.New(
    gateway.WithAPIKey("TELNYX_API_KEY"),
    gateway.WithPhoneNumber("+15551234567"),
    gateway.WithConnectionID("your-connection-id"),
    gateway.WithSTTProvider("deepgram", "DEEPGRAM_API_KEY"),
    gateway.WithLLMProvider("anthropic", "ANTHROPIC_API_KEY"),
    gateway.WithTTSProvider("elevenlabs", "ELEVENLABS_API_KEY"),
)

// Start HTTP server for webhooks and WebSocket
gw.ListenAndServe(ctx, ":8080")
```

Features:

- **STTв†’LLMв†’TTS Pipeline**: Real-time voice processing chain
- **Voice Activity Detection**: Energy-based Ој-law speech detection
- **Barge-in Support**: Immediate, after_sentence, or disabled modes
- **Provider Factory**: Deepgram/OpenAI STT, Anthropic/OpenAI LLM, ElevenLabs/OpenAI/Deepgram TTS
- **Tool Calling**: LLM function calling with configurable handlers
- **Session Metrics**: Track turn count, interruptions, speech duration

### MMS/RCS Messaging

Extended messaging support beyond SMS:

```go
import "github.com/plexusone/omni-telnyx/omnivoice/callsystem"

// Unified SendMessage API
msg, err := provider.SendMessage(ctx, &callsystem.SendMessageParams{
    To:        "+15559876543",
    From:      "+15551234567",
    Body:      "Check out this image!",
    MediaURLs: []string{"https://example.com/image.jpg"}, // MMS
})

// Convenience methods
provider.SendMMS(ctx, "+15559876543", "Hello!", []string{"https://example.com/image.jpg"})
provider.SendMMSFrom(ctx, to, from, body, mediaURLs)

// RCS via Messaging Profile
msg, err := provider.SendMessage(ctx, &callsystem.SendMessageParams{
    To:                  "+15559876543",
    Body:                "RCS message with fallback",
    MessagingProfileID:  "your-messaging-profile-id",
})
```

### OmniChat Provider

New `provider.Provider` implementation for SMS/MMS/RCS:

```go
import "github.com/plexusone/omni-telnyx/omnichat"

p, err := omnichat.New(
    omnichat.WithAPIKey("TELNYX_API_KEY"),
    omnichat.WithPhoneNumber("+15551234567"),
    omnichat.WithMessagingProfileID("profile-id"), // For RCS
)

// Connect and handle messages
p.Connect(ctx)
p.OnMessage(func(ctx context.Context, msg provider.IncomingMessage) error {
    // Handle incoming SMS/MMS/RCS
    return p.Send(ctx, msg.ChatID, provider.OutgoingMessage{
        Content: "Thanks for your message!",
    })
})

// Mount webhook handler
http.Handle("/webhook", p.WebhookHandler())
```
