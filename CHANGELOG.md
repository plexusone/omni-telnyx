## [v0.3.0] - 2026-06-13

### Highlights

- Full voice gateway for Telnyx Media Streaming with STT→LLM→TTS pipeline

### Added

- Voice gateway implementing `coregateway.Gateway` interface with HTTP/WebSocket server ([`72c63bc`](https://github.com/plexusone/omni-telnyx/commit/72c63bc))
- Session type implementing `coregateway.Session` interface with metrics tracking ([`72c63bc`](https://github.com/plexusone/omni-telnyx/commit/72c63bc))
- JSON webhook handlers for Telnyx call events (`call.initiated`, `call.answered`, `call.hangup`) ([`72c63bc`](https://github.com/plexusone/omni-telnyx/commit/72c63bc))
- WebSocket connection wrapper for Telnyx Media Streaming protocol ([`72c63bc`](https://github.com/plexusone/omni-telnyx/commit/72c63bc))
- Voice pipeline with STT→LLM→TTS processing chain ([`72c63bc`](https://github.com/plexusone/omni-telnyx/commit/72c63bc))
- Voice activity detection and silence-based end-of-speech detection ([`72c63bc`](https://github.com/plexusone/omni-telnyx/commit/72c63bc))
- Barge-in interruption support (immediate, after_sentence, disabled modes) ([`72c63bc`](https://github.com/plexusone/omni-telnyx/commit/72c63bc))
- Provider factory for STT (Deepgram, OpenAI), LLM (Anthropic, OpenAI), TTS (ElevenLabs, OpenAI, Deepgram) ([`72c63bc`](https://github.com/plexusone/omni-telnyx/commit/72c63bc))
